# March Madness Simulation

A project to simulate and predict NCAA March Madness brackets using historical and current data.

## Structure
- `data/`: Data files (raw and processed)
- `src/`: Source code (simulation, models, etc.)
- `notebooks/`: Jupyter notebooks for exploration and analysis

## Getting Started
1. Install dependencies: `pip install -r requirements.txt`
2. Run simulations from `src/main.py`
